# Contributing

Thank you for considering contributing to this project.

## How to Contribute

1. Fork the repository.
2. Create a new branch.
3. Make a focused change.
4. Test the documentation or template.
5. Open a pull request.

## Contribution Rules

- Keep the guide safe and educational.
- Do not add bypass, spoofing, MITM, subscription manipulation, or payment-related instructions.
- Keep source credit intact.
- Use clear Markdown formatting.
