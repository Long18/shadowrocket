# Security Policy

## Reporting a Vulnerability

If you find a security issue, please do not open a public issue.

Contact the maintainer privately.

## Scope

This project only provides documentation and configuration templates for personal DNS setup.

Do not submit or request:

- Subscription bypass
- Entitlement spoofing
- Payment manipulation
- MITM certificate instructions
- Credential theft
- Malware
- Phishing
